### 🧛 Gastos Vampiro - Versión Final (Ejecutable)

¡Bienvenido al lanzamiento oficial de **Gastos Vampiro**! Este proyecto es el resultado práctico de nuestro **Curso Maestro de Python GUI con Tkinter**, donde aprendimos a construir una aplicación real desde cero hasta su distribución.

#### 📝 Sobre este programa
Este software ha sido diseñado con **fines puramente educativos** para demostrar el potencial de Python y la librería Tkinter en el desarrollo de interfaces gráficas de escritorio. La aplicación permite gestionar suscripciones mensuales, calcular gastos anuales y exportar datos.

#### 🚀 Instrucciones de Ejecución
Para probar la aplicación en tu computadora con Windows, sigue estos pasos:

1. **Descarga el archivo:** Descarga el archivo comprimido `gastos_vampiro.zip` que se encuentra en los *Assets* aquí abajo.
2. **Descomprime:** Haz clic derecho sobre el archivo descargado y selecciona "Extraer todo".
3. **Ejecuta:** Entra en la carpeta extraída y busca la ruta:
   `gastos_vampiro/gastos_vampiro.exe`
4. **Prueba el resultado:** Haz doble clic en el archivo `.exe` para iniciar la aplicación.

#### 🛠️ Detalles Técnicos
* **Lenguaje:** Python 3.x
* **Librería GUI:** Tkinter / TTK (Temas modernos)
* **Compilación:** Generado mediante **PyInstaller**.
* **Persistencia:** Manejo de datos locales a través de archivos JSON.

#### ⚠️ Nota de Seguridad (Falso Positivo)
Debido a que el ejecutable ha sido generado con PyInstaller y no cuenta con una firma digital de desarrollador corporativo, es posible que **Windows Defender o SmartScreen** muestren una advertencia al intentar ejecutarlo. 
* Esto es un **falso positivo** común en aplicaciones creadas con Python. 
* Puedes ejecutarlo con total seguridad haciendo clic en *"Más información"* y luego en *"Ejecutar de todas formas"*. El código fuente completo está disponible en este repositorio para tu revisión.
